# Hanging Game

Small, local hangman game built with React + TypeScript + Vite based on the course project by Fernando Herrera progra-101.

Usage
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`

How to play
- Guess letters to reveal the hidden word. Wrong guesses reduce your lives; the game ends when the word is solved or lives run out.

Files
- Source: `src/`
- Entry: `src/main.tsx`
- Game logic: `src/helpers/`
